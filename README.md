# Ecronx Technologies

A full-stack application with React frontend and Django backend.

## Project Structure

```
Ecronx-Technologies/
├── frontend/     # React application
├── backend/      # Django REST API
├── README.md
├── .gitignore
├── package.json
├── requirements.txt
└── .env.example
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```

## Environment Setup

Copy `.env.example` to `.env` and update with your configuration.

```bash
cp .env.example .env
```
